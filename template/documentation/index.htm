Getting Started

For installing this template, there are a few requirements you need on your web server in order to get full functionality. You will need to contact your webhost if the following requirements are missing.

  • PHP 5.2+
  • PHP Mail Support

You must have JavaScript enabled in your browser. Also, a web server is required for the video background - testing it locally (unless using a development server such as Wamp or Mamp server) will not work.

If a user does not have JavaScript enabled in their browser, the site does not load and a message is displayed informing them of the problem.

This is not a Wordpress theme, so cannot be installed with Wordpress.

Installation

To install 'Professional' on your web server, download the files from Themeforest and unzip them to the local disk on your computer. Inside the extracted folder you will see two folders named Documentation and HTML . Using an FTP Client (such as Filezilla or Total Commander), upload all of the contents from your specific folder of the HTML folder to your server.

Youtube Background

Open up the yt-video/index.html file, and look for the video background section with property value. 

	
<!-- YouTube background video -->
<a id="bg-video" class="player" data-property="{videoURL:'https://www.youtube.com/watch?v=dbFV2CqJQ6U',containment:'body',autoPlay:true, mute:true, startAt:0, showControls:false}" data-poster="images/videoBg.jpg">youtube</a>

Once you have chosen your YouTube video, copy the video URL of the video - don't copy the embed code as this will not work. Paste this address into the videoURL. You should see the video on the page. Note that the videos can take a while to load sometimes depending on your connection, and the loading overlay shows until the video is ready to play, hence why it sometimes loads for quite a while.

Note that the video background only works on web servers. If testing locally, you will see a constant loading screen.

The data-poster parameter need to set the image. When any device not support the video then the background image will show.

Contact Form

The contact form supplied with this template works pretty much on its own, all you need to do is set up who the messages are emailed to along with a few other details.

To set up the contact form, you need to supply three pieces of information. Most importantly is your email address, which is where any messages from the contact form are sent to. Secondly, you need to give your personal or company name, which should be the name associated with your email account which is receiving the emails. This helps stop your messages being marked as spam. Thirdly, you need provide the name of your site which will show email clients where the email is from - helping stop your emails being marked as spam.

If you're not receiving any emails, check they're not being blocked by your spam filter.

To set up the contact form, open up sendemail.php and look for the following section of three variables, near the top of the file: 

    $to = 'saif.shajib@gmail.com';  // Change it by your email address
    $yourname ='Professional';
    $siteEmail = <info@yoursite.com>;

To enter your email address, type it in between the speech marks (' ') in the variable $to and do the same for the next two variables for changes with appropriate value.

All settings must be in speech marks, like this: ' ... '

If you want to customise the error and success messages of the contact form, you can find below of this file. 

<div class="content-message"><i class="fa fa-rocket"></i> Email Sent Successfully</div>
<div class='content-message'><i class='fa fa-times'></i> An error has occured. Please try later.</div>

To change the messages, simply edit the text (Any error can fail to send email.).

Antispam code are integrated there, so not necessary to add captcha.
